## Frequently Asked Questions

### I did not receive my API key by email, what should I do?

New accounts are typically activated within 5 minutes of subscribing, if you did not receive your API key by email then you can always see your current API key by pressing the "Login" button on the [RatingPosterDB Website](https://ratingposterdb.com).

### I changed my settings but the images did not change in the app I use

After changing your image settings on the [manager page](https://manager.ratingposterdb.com) it can take up to 3 minutes for the changes to apply, but the image cache of the app you use can take significantly longer to expire. If you want to see your changes applied fast we suggest that you use the "Login" button on the [RatingPosterDB Website](https://ratingposterdb.com) and reset your API key from there, then uninstall any addons/plugins you are using and install them again with the new key, this will invalidate the image cache from the app so you should see your settings applied immediately.

### How do I upgrade to lifetime?

To upgrade to lifetime from a monthly subscription is easy, simply go to [this page](https://www.patreon.com/settings/memberships/rpdb) and choose "Switch to another tier". After receiving your lifetime API key by email you should cancel membership from [this page](https://www.patreon.com/settings/memberships/rpdb).

To upgrade to lifetime from a yearly subscription though is more complicated, the only way we know of doing this with Patreon is by creating a new Patreon account by adding a dot "." somewhere in your email (example: "al.ex@gmail.com" instead of "alex@gmail.com"), email providers ignore dots, so the emails are sent to the same inbox, but Patreon will consider it a new email so you can create another account with it. You should then subscribe to lifetime from the new account (and unsubscribe after receiving your API key) and send us a message on Patreon to refund the payment from the initial account.

### How do I cancel the subscription?

To unsubscribe you should do so from Patreon, specifically this [Patreon RPDB Membership](https://www.patreon.com/settings/memberships/rpdb) page, there is a "Cancel membership" button on this page.
